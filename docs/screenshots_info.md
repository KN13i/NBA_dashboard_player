# Screenshots du Dashboard

Ce dossier doit contenir les captures d'écran du dashboard Power BI pour illustrer son utilisation.

## Screenshots recommandés

### 1. dashboard_overview.png
- **Description** : Vue d'ensemble de la page principale
- **Contenu** : Affichage complet du dashboard avec les statistiques principales
- **Utilisation** : Page d'accueil, README principal

### 2. dashboard_filters.png
- **Description** : Démonstration des filtres interactifs
- **Contenu** : Panneau de filtres avec plusieurs options sélectionnées
- **Utilisation** : Guide d'utilisation, section filtres

### 3. dashboard_player_comparison.png
- **Description** : Comparaison entre plusieurs joueurs
- **Contenu** : Graphiques comparatifs (radar chart, barres, etc.)
- **Utilisation** : Démonstration des capacités d'analyse

### 4. dashboard_team_stats.png
- **Description** : Statistiques par équipe
- **Contenu** : Vue des performances par équipe NBA
- **Utilisation** : Analyse d'équipe

### 5. dashboard_performance_trends.png
- **Description** : Tendances de performance
- **Contenu** : Graphiques d'évolution dans le temps
- **Utilisation** : Analyse temporelle

## Format des screenshots

- **Format** : PNG (pour une meilleure qualité)
- **Résolution** : Minimum 1920x1080 pixels
- **Qualité** : Haute définition
- **Annotations** : Ajouter des flèches ou légendes si nécessaire

## Comment prendre les screenshots

1. Ouvrir le dashboard dans Power BI Desktop
2. Mettre en mode plein écran (F11)
3. Utiliser l'outil de capture Windows (Win + Shift + S) ou équivalent
4. Sauvegarder avec un nom descriptif
5. Placer dans ce dossier `docs/`

## Exemple de nommage

```
docs/
├── screenshot_01_vue_ensemble.png
├── screenshot_02_filtres_actifs.png
├── screenshot_03_comparaison_joueurs.png
├── screenshot_04_statistiques_equipes.png
├── screenshot_05_tendances_performance.png
└── screenshot_06_details_joueur.png
```
